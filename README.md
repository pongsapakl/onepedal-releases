## What is 1P?

1P is a free NLP-controlled guitar effects plugin. Describe your tone in plain language ("warm blues crunch", "shoegaze reverb") instead of tweaking knobs.

- **VST3 + AU** — works in Ableton, Logic Pro, Reaper, FL Studio, and more
- **Standalone app** — no DAW required
- **macOS** — Windows coming soon
